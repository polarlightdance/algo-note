# 初识算法

1. 算法是什么？
    算法是在有限时间内解决特定问题的一组指令或操作步骤，具有以下特点
    
    - 问题是明确的，包含清晰的输入和输出定义
    - 具有可行性，能够在有限步骤、时间和内存空间下完成
    - 各步骤都有确定的含义，在相同的输入和运行条件下，输出始终相同

2. 数据结构的定义
    数据结构是组织和存储数据的方式，涵盖数据内容、数据之间关系和数据操作方法

    - 空间占用尽量少，以节省计算机内存
    - 数据操作尽可能快速，涵盖数据访问、添加、删除、更新等
    - 提供简洁的数据表示和逻辑信息，以便算法高效运行
